# Betrayal / Revenge — Eligible Template Vocabulary

These templates are **research-derived** title and opening-line patterns for the
betrayal/revenge emotional lane on Reddit Stories. The script generation
prompt (`prompts/reddit_generation_prompt.txt`) references this file as the
eligible vocabulary for adaptation — not as text to copy. Each generated
story should ADAPT a pattern from the relevant pool, never quote it verbatim.

## Usage notes for the prompt
- Pick ONE title template per script and adapt it to the chosen subgenre.
- Pick ONE opening template — DO NOT quote it verbatim, ADAPT the structure.
- The bureaucratic-revenge climax requirement applies regardless: the catharsis
  is calm precision, not confrontation.
- Validate the promise system: primary title, thumbnail text, and first 30
  seconds must communicate the SAME single emotional promise.

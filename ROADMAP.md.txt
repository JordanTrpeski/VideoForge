# VideoForge Roadmap

A multi-channel AI video production system. Pulls topics, generates
scripts with Claude, narrates with TTS, assembles with MoviePy,
captions with faster-whisper, and uploads to YouTube / TikTok /
Instagram. Reviewed and approved via web dashboard.

## Phase 13 — Multi-channel readiness

**Goal:** finish VideoForge so it can run three production channels
(Reddit Stories, Dark Psychology, Sleep Lore) with cross-platform
distribution and remote review from work.

### Block A — Video templates per channel
Per-channel content templates. Each defines: visual_mode, length
range, hook style pool, music palette, thumbnail mode, prompt
overrides, dual_output flag. Variation engine picks template per job,
then varies length and hook within that template's pool.

### Block B — API usage tracking
Per-channel, per-provider tracking of units used and estimated cost.
Dashboard page with month-to-date breakdown and 80% budget alerts.
Nightly rollup. Providers: claude, elevenlabs, openai_tts, kokoro,
leonardo, youtube, reddit, tiktok, instagram, r2.

### Block C — Long-form ambient visual mode
New visual_mode 'long_form_ambient' for sleep content. Looped ambient
clip up to 3 hours, optional overlay image, captions-off support,
mixed ambient audio at configurable dB. Variation engine length cap
raised.

### Block D — TikTok + Instagram upload
Cross-platform funnel for teaser shorts. After YouTube short upload
succeeds, schedule parallel TikTok + Instagram uploads 6 hours later
with YouTube long-form URL injected into descriptions. Per-channel
platform enable/disable flags.

### Block E — Channel-aware editors
Config and prompt editors write to channels/<slug>/ overlay when a
channel is selected, fall back to global only when no channel filter
is active. UI badge indicates scope.

### Block F — Cloud video preview (Cloudflare R2)
After captioning, upload final video and thumbnail to R2. Dashboard
plays from preview_url. Enables review from any device, any network,
without home upload bandwidth bottleneck. Local fallback if R2
upload fails.

### Block G — R2 lifecycle and retention
Per-channel retention_days (default 7) and
keep_after_youtube_upload (default false). Nightly cleanup job.
Dashboard shows R2 storage used per channel and next-cleanup date.

### Discipline rules
- Each block completes with explicit pass conditions before the next
  begins.
- After each block: brief summary of files touched and pass condition
  result.
- No scope expansion mid-build. Anything that comes up goes in
  PHASE_14_NOTES.md instead.
- Final step is read-only audit against all seven blocks with file
  path and line number evidence per item.

## Channels — strategy

### Channel 1: Reddit Stories
First-person AI-written drama fiction in Reddit style. 20–40 min
long-form + 45–60s teaser shorts. Female Kokoro voice, locked.
Mixed backgrounds: gameplay, satisfying content, ambient. Cross-
platform to TikTok and Instagram Reels.

### Channel 2: Dark Psychology / Philosophy
Video essay format on Mindplicit / Dark Psychology Coded model.
8–15 min long-form. Male Kokoro voice, calm and intellectual.
Leonardo cinematic imagery. 70/30 mix of hard-edge and introspective
content. Light teaser shorts.

### Channel 3: Sleep Lore
Pure mythology and folklore long-form ambient. 1–3 hour episodes.
Neutral calm Kokoro voice. Looped fireplace / forest / rain visuals.
No captions. Public domain source material. Podcast distribution
(Spotify, Apple Podcasts) instead of TikTok funnel.

### Launch sequence
1. Phase 13 build completes (this phase)
2. Dry run channel 1 on existing Engineering Brief channel
3. Launch channel 1 (Reddit Stories) on fresh Google account
4. Run 14–21 days, daily editorial gate
5. If v30 metrics green: launch channel 2 (Dark Psychology)
6. Same gap, then launch channel 3 (Sleep Lore)

### Identity / account strategy
Setup C — separate Google account, separate AdSense, separate
TikTok and Instagram per channel. Engineering Brief used as dry-run
target only, then private and abandon.

## Operating principles

- Editorial gate stays real per channel. If it slips, kill the
  weakest channel before scaling.
- Kill metrics calibrated to runway and time budget. Day 60 review
  is hard, on the calendar, non-negotiable.
- AdSense linkage risk: separate Google accounts per channel to
  isolate. One strike does not cascade.
- YouTube inauthentic content policy is the existential risk.
  Defense is editorial fingerprint per channel: distinct voice,
  distinct visual language, distinct content category, human
  judgment on every video.
- Cost discipline: ~$10–25/month total across all three channels.
  R2 free tier with 7-day retention. Kokoro instead of paid TTS at
  long-form lengths.
